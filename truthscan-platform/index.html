How Image Forensics Works
01
ELA — Error Level Analysis
Re-encodes the image at a fixed quality and amplifies the difference. Edited regions have different compression error levels than untouched areas, revealing manipulation zones.
02
EXIF Metadata Extraction
Reads embedded camera metadata — device model, GPS, timestamps, software. Flags suspicious entries like editing software fingerprints, mismatched timestamps, or stripped metadata.
03
Noise & Compression Analysis
Detects inconsistent noise grain and JPEG block artifacts across image regions. Spliced or cloned areas typically show different noise characteristics than the native image.
04
Lighting & GAN Forgery
Analyzes shadow directions, highlight coherence, and color temperature consistency. GAN detection flags frequency-domain artifacts and pixel-level patterns unique to AI-generated content.

How Content DNA Works
01
Upload Content
Upload an image, paste text, or submit a video. The system extracts features for fingerprinting regardless of format or quality.
02
Generate Embedding
CLIP generates 768-dim image vectors; Sentence-Transformers produce 384-dim text vectors. A perceptual hash captures pixel-level features.
03
Vector Search
The embedding is compared against all stored fingerprints via FAISS cosine similarity. Combined with pHash distance for images for accuracy.
04
Classification
>90% = Duplicate, 70-90% = Modified, 50-70% = Similar, <50% = Original. Results include top matches, scores, and metadata.

Platform Architecture

Built for
Truth &
Transparency

TruthScan is a two-module platform combining probabilistic AI detection with real-time fact verification powered by Claude AI and live web search.

Platform Modules
01
AI & Deepfake Detector
Simulated multi-model pipeline analyzing GAN fingerprints, diffusion artifacts, perplexity, and burstiness across images, video, text, and documents.
02
Fake News Verifier
Real AI-powered module using Claude Sonnet + live web search to extract claims, verify against 10+ authoritative sources, and return structured verdicts.
03
Image Originality Detector
Real canvas-based ELA using browser APIs. Claude AI analyzes pixel statistics, EXIF metadata, and noise patterns to detect tampering, splicing, copy-move, and GAN forgery.
04
Ethics & Transparency
Results are clearly labeled as probabilistic estimates. The system never fabricates sources, always shows evidence, and returns "Inconclusive" when data is insufficient.
